取得のヒント（各OS）

以下の方法でソフト名とバージョンを取得し、本ツールに貼り付けて整形してください。

Windows

  • winget: winget list

Linux

  • Deb系: dpkg -l
  • RPM系: rpm -qa
  • Flatpak/Snap: flatpak list / snap list

macOS

  • Homebrew: brew list --versions
  • 全アプリ: system_profiler SPApplicationsDataType
